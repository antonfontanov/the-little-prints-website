
  # The Little Prints

  A modern React website for children's art posters with custom sizing and ordering system. This project creates a beautiful, responsive website for selling customizable children's art prints.

  ## Features

  - Modern React + TypeScript frontend
  - Tailwind CSS for styling with shadcn/ui components
  - Responsive design optimized for all devices
  - Custom ordering system with size and quantity selection
  - Azure Static Web Apps deployment ready

  ## Getting Started

  ### Prerequisites
  - Node.js 18 or higher
  - npm or yarn

  ### Installation

  1. Clone the repository:
  ```bash
  git clone https://github.com/antonfontanov/the-little-prints-website.git
  cd the-little-prints-website
  ```

  2. Install dependencies:
  ```bash
  npm install
  ```

  3. Start the development server:
  ```bash
  npm run dev
  ```

  4. Open [http://localhost:5173](http://localhost:5173) in your browser.

  ## Deployment

  This project is configured for automatic deployment to Azure Static Web Apps via GitHub Actions. Simply push to the main branch to trigger a deployment.
